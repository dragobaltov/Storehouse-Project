◆ add()

void StorehouseEngine::add ( )
private

Gets data from the user to create a Product object. If the product is already in the storehouse, increases its quantity with the given. If not, asks the user to input place data. If the input is not correct(the place is occupied or invalid), the user should input new data until it is valid. Finnaly, adds the product at the given place.

Note
All products are added with current date.
